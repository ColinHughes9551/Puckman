Puckman! - A mostly faithful recreation of Pac Man with the addition of portals from the Portal series
Created by Colin Hughes for CPSC 386

--INSTRUCTIONS
Requires Python 3.7 and Pygame
Execute by running main.pyw
If experiencing difficulties, try instead executing main.pyw or game.py from within a python IDE

--CONTROLS
Arrow keys to move
Z to fire the blue portal
X to fire the orange portal
Q to quit the game
